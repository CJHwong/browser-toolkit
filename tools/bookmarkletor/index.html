What is a Bookmarklet?

A bookmarklet is a small piece of JavaScript code saved as a browser bookmark. When you click it, the code runs on the current web page, allowing you to add new features or modify its behavior.

How to Install

  1. Generate the bookmarklet using the tool below.
  2. A link will appear that says "Drag Me to Bookmarks".
  3. Click and drag this link to your browser's Bookmarks Bar.
  4. Click the new bookmark on any page to run your code.

Tip: If your Bookmarks Bar is hidden, press Cmd/Ctrl + Shift + B to show it.
